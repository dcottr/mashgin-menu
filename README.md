# Mashgin Menu

This is built using [T3 Stack](https://create.t3.gg/) and

- [Next.js](https://nextjs.org)
- [Prisma](https://prisma.io)
- [tRPC](https://trpc.io)

# Building Locally

Make sure you've installed nvm, bun, and Docker.

```
$ nvm install v22.13.1
$ nvm use v22.13.1
$ bun install
$ ./start-database.sh
$ bun run dev
```
